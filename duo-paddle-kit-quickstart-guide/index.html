Duo Paddle Kit: Quickstart Guide

Thank you for your purchase of the Duo Paddle Kit: An simple do it yourself paddle kit.

To get started, please watch this build video:

Some high level tips:

  • Installing the levers onto the base is normally a one time thing. If you need to remove them, wiggle them out while pulling straight up to not brake the pivot points.
  • Installing the top onto the base is normally a one time thing. If you need to remove it, please watch the video for a tip on how to pull on the front of the case to create a small gap and use fingernails to wiggle walk the top off.
