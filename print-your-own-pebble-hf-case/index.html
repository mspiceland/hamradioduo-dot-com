Print Your Own Pebble HF Case

May 22, 2026

Here are the files to print your own case for the Pebble HF. You can print them in any color you choose, add your callsign, or modify it.

If you have a Bambu printer, you can use the .3mf file. It already has the .stl files positioned correctly, including the lettering embedded in the top face down for easy printing. If you need the .stl files for another printer, they are also attached below.

Printing Instructions

We use PETG-HF for both the case and the knob.

For Bambu printers, you should simply have to open the .3mf file, change any filaments to match yours, and print. It will prompt you if you have a different printer. For other printers, you will need to import the .stl files. For Bambu, we import the top and lettering file at the same time and it combines them. We place it face down and it barely uses any extra filament for the tower from being two colors.
